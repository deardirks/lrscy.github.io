前言

TensorFlow安装环境为Ubuntu 16.04.3 LTS，GPU为GT 750M。

假设目前已经安装好了CUDA8，如果没有安装请依照「Ubuntu 16.04下CUDA Tookit 8安装」进行安装。

如果有一定英语能力的同学最好请移步官网进行下载安装，尽管可能需要下科学上网。
